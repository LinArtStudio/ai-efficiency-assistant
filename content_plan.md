# B站内容选题与模板

## 平台策略

### 为什么选择B站
- **数据来源**：行业公开数据（B站2025年商业生态报告）
- **核心依据**：
  - B站科技区日均播放量增长47%，"效率工具"类内容增速最快
  - 18-35岁用户占比78%，与目标用户（职场新人、中层管理者）高度重合
  - "教程+实测"类视频完播率比纯介绍类高2.3倍

### 内容定位
- **人设**：在校大学生开发者，分享用AI解决办公痛点的实战经验
- **风格**：真实、接地气、有数据支撑，避免过度营销
- **更新频率**：每周2-3条，前两周密集发布建立基础

## 半自动化脚本生成说明

### 脚本生成逻辑
使用 `content_batch.py` 工具，输入以下参数即可生成脚本初稿：
- `title`：视频标题
- `hook`：钩子脚本
- `pain_point`：核心痛点描述
- `solution`：解决方案要点
- `data_source`：数据来源说明

### 生成示例
```bash
python content_batch.py \
  --title "周报写到吐？我用AI一键生成，主管还夸我写得详细" \
  --hook "每周五下午3点，全办公室都在憋周报...直到我发现了这个方法" \
  --pain_point "72%的职场人讨厌写周报，平均耗时1.5小时/周" \
  --solution "AI自动润色、补充数据、生成专业周报" \
  --data_source "行业调研+用户痛点"
```
