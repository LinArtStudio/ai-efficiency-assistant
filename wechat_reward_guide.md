# 微信赞赏码创建指南

## 步骤1：打开微信收付款

1. 打开微信
2. 点击右上角「+」→「收付款」
3. 或者：「我」→「支付」→「收付款」

## 步骤2：生成赞赏码

1. 在收付款页面，找到「赞赏码」
2. 点击「赞赏码」
3. 如果没有，点击「更多」→「赞赏码」

### 设置赞赏金额
- **建议金额**：¥29（与 Pro 版价格一致）
- **赞赏语**：「AI办公效率助手 - Pro版开通」

## 步骤3：保存赞赏码

1. 点击「保存赞赏码」
2. 保存到手机相册
3. 发送到电脑备用

## 步骤4：替换网站中的占位图

### 方法1：直接替换图片
1. 将赞赏码图片上传到 GitHub 仓库
2. 修改 `index.html` 中的图片路径

```html
<!-- 找到赞赏码占位区域 -->
<img src="你的赞赏码图片路径.png" alt="微信赞赏码" style="width: 200px;">
```

### 方法2：使用图床
1. 上传到免费图床（如 SM.MS）
2. 获取图片链接
3. 替换 `src` 属性

```html
<img src="https://你的图床链接.png" alt="微信赞赏码">
```

## 步骤5：添加支付说明

在赞赏码旁边添加说明：

```html
<div class="payment-info">
  <h3>微信赞赏码支付</h3>
  <p>1. 扫描赞赏码支付 ¥29</p>
  <p>2. 截图发送至公众号/私信</p>
  <p>3. 24小时内开通 Pro 权限</p>
  <p style="color: #666; font-size: 0.9em;">
    💡 备注「Pro升级」，方便快速处理
  </p>
</div>
```
