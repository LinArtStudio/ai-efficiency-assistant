# B 站视频发布检查清单

> 每次发布视频前，逐项检查。✅ 已完成 / ⬜ 未完成

---

## 一、录制前准备

- [ ] 脚本已确认（使用优化版脚本）
- [ ] 录屏软件已安装（推荐 OBS Studio，免费）
- [ ] 麦克风已测试，音量正常
- [ ] AI 工具已登录，功能可用
- [ ] 演示用的会议录音/文本已准备
- [ ] 桌面清理干净，无关窗口已关闭
- [ ] 通知已关闭（避免弹窗干扰录屏）

## 二、录制

- [ ] 分段录制（按脚本章节），方便后期剪辑
- [ ] 每段开头留 2 秒空白，方便剪辑衔接
- [ ] 操作演示要慢，鼠标移动清晰
- [ ] 关键步骤停留 3-5 秒，让观众看清
- [ ] 录制完检查：音频是否正常、画面是否清晰

## 三、剪辑

- [ ] 添加字幕（B站自动字幕 + 手动校对）
- [ ] 关键数据用字幕特效强调（如 "2小时→3分钟"）
- [ ] 添加 BGM（轻音乐，音量 10-15%）
- [ ] 片头 3 秒：标题 + 钩子
- [ ] 片尾 5 秒：CTA + 下期预告
- [ ] 总时长控制在 5-6 分钟
- [ ] 导出格式：1080p，mp4

## 四、发布设置

- [ ] 标题已填写（使用发布素材中的标题）
- [ ] 封面已制作（16:10 比例，1920x1200）
- [ ] 简介已填写（包含 CTA + 工具链接 + 标签）
- [ ] 标签已添加（至少 5 个相关标签）
- [ ] 分区选择：知识 > 职场技能
- [ ] 定时发布：工作日 19:00-21:00

## 五、发布后

- [ ] 评论区置顶：工具链接 + 使用说明
- [ ] 朋友圈分享
- [ ] B站动态发布（图文版精华摘要）
- [ ] 记录到内容追踪表（更新发布状态）
- [ ] 24 小时后查看数据（播放量、完播率、互动率）
