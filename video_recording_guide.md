# B站视频录制指南

## 📹 视频 #01：周报写到吐？

### 录制清单

#### 1. 痛点场景录制（1-2分钟）

**场景设置**：
- 桌面：打开一个空白 Word 文档
- 屏幕录制：使用 OBS 或手机支架
- 环境：安静、光线充足

**表演要点**：
- 表现「痛苦」表情（抓头、叹气、看时间）
- 边打字边自言自语：「这周干了啥来着？」
- 显示时间：「已经写了1小时了...」

**录制脚本**：
```
[画面：屏幕录制 - 空白 Word 文档]

旁白：「每周五下午三点，办公室会出现一个神奇的现象——
所有人都在敲键盘，但没人知道在写什么。

没错，都在憋周报。

我也是，看看现在几点了...下午4点半，
已经写了1小时了，才写了一半。」
```

#### 2. 工具演示录制（2-3分钟）

**操作步骤**：
1. 打开 https://linartstudio.github.io/ai-efficiency-assistant/tool.html
2. 点击「周报生成」标签
3. 输入示例内容：
   ```
   周一：和客户开会对需求
   周三：完成方案初稿
   周五：提交修改版
   ```
4. 点击「生成」按钮
5. 展示生成结果
6. 复制到剪贴板

**录制要点**：
- 操作要慢，让观众看清
- 关键步骤放大（Ctrl + 滚轮）
- 加入旁白解释

#### 3. 效果对比（30秒）

**画面分割**：
- 左边：手写周报（痛苦、耗时）
- 右边：AI 生成（快速、专业）

**数据展示**：
- 手写：1.5 小时
- AI 生成：5 分钟
- 节省：1 小时 25 分钟

#### 4. 结尾引导（30秒）

**话术**：
```
「想要体验的，评论区扣1，私信我发链接。

免费版够用，不用有心理负担。

前100名用户享受永久免费个人版权限。

我是LinArtStudio，一个用AI解决办公痛点的在校生，
我们下期见。」
```

## 📹 视频 #02：会议记录整理

### 录制清单

#### 1. 痛点场景录制（1分钟）

**场景设置**：
- 模拟会议场景（可以找室友帮忙）
- 显示「会议进行中」
- 会后显示「整理会议记录」

**表演要点**：
- 会议中：认真记录、频繁暂停录音
- 会后：痛苦整理、反复听录音

**录制脚本**：
```
[画面：会议场景]

旁白：「作为项目经理，我最怕的不是开会。
是开完会之后，还要花1.5小时整理会议记录。

3场会下来，半天就没了。」
```

#### 2. 工具演示录制（2-3分钟）

**操作步骤**：
1. 打开 tool.html
2. 点击「会议纪要」标签
3. 输入示例内容：
   ```
   张三：这周的接口需要在周三前完成联调
   李四：UI设计稿今天会出，确认后可以开始切图
   王五：决定采用方案A，下周一开始开发
   赵六：需要确认第三方API的接口文档
   ```
4. 点击「生成」按钮
5. 展示生成结果

**录制要点**：
- 强调「自动提取」功能
- 展示「待办事项」和「决策」
- 演示「复制到剪贴板」

#### 3. 用户反馈（30秒）

**展示内容**：
- 截图：用户反馈（可以找室友录制）
- 数据：节省时间统计

**话术**：
```
「这是早期用户的反馈。

项目经理小王说：以前最怕开完会还要整理记录，
现在反而期待开会了，因为5分钟就搞定。」
```

## 🎬 录制设备

### 必备设备
- **电脑**：用于屏幕录制
- **手机**：用于真人出镜（可选）
- **麦克风**：手机自带即可，或用耳机麦

### 推荐软件
- **屏幕录制**：OBS Studio（免费）
- **视频剪辑**：剪映（免费、易用）
- **字幕添加**：剪映自动识别

### 录制环境
- **光线**：自然光或台灯，避免逆光
- **背景**：简洁干净，避免杂乱
- **声音**：安静环境，避免回音

---

## 🎬 剪辑要点

### 节奏控制
- **前3秒**：必须抓人（痛点钩子）
- **每30秒**：一个信息点
- **结尾**：明确引导（扣1、私信）

### 字幕添加
- 使用剪映自动识别
- 关键词加粗或变色
- 数据部分单独展示

### 封面制作
- 尺寸：1146×717 像素
- 内容：痛点 + 数据 + 人物
- 风格：对比强烈、文字清晰

## 💡 录制技巧

### 真人出镜
- 看镜头，不要看屏幕
- 语速适中，不要太快
- 表情自然，适当手势

### 屏幕录制
- 关闭通知弹窗
- 隐藏桌面图标
- 放大关键操作

### 声音录制
- 保持嘴离麦克风 15-20cm
- 避免喷麦（可用纸巾挡）
- 后期可降噪
