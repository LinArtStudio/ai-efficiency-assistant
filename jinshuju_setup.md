## 🚀 立即设置（15分钟）

### 步骤1：注册金数据（3分钟）

1. 打开 https://jinshuju.net
2. 点击「免费注册」
3. 使用手机号或邮箱注册
4. 登录

### 步骤2：创建表单（5分钟）

1. 点击「创建表单」
2. 选择「空白表单」
3. 表单名称：`AI办公效率助手 - 用户注册`

### 步骤3：添加字段（5分钟）

按顺序添加以下字段：

| 字段类型 | 字段名称 | 是否必填 | 选项 |
|---------|---------|---------|------|
| 姓名 | 姓名 | ✅ 必填 | - |
| 邮箱 | 邮箱 | ✅ 必填 | - |
| 单选 | 行业 | ✅ 必填 | 互联网/科技、制造业、金融、教育、医疗、咨询、政府/国企、创业、自由职业、在校学生、其他 |
| 单选 | 职位 | ✅ 必填 | 项目经理/PMO、部门主管/总监、产品经理、运营/市场、HR/行政、技术/开发、创业者/企业主、自由职业者、在校学生、其他 |
| 多行文本 | 你最希望AI帮你解决什么问题？ | 选填 | - |

### 步骤4：设置表单（2分钟）

1. 点击「设置」
2. 开启「提交后显示感谢页面」
3. 感谢页面内容：
   ```
   ✅ 注册成功！

   感谢注册 AI办公效率助手！

   🎁 你的专属福利：
   - 前100名用户享受永久Pro权限
   - 工具链接：https://你的Gitee用户名.gitee.io/ai-efficiency-assistant/tool.html

   遇到问题？邮件联系：15902234202@163.com
   ```

### 步骤5：获取表单链接（1分钟）

1. 点击「发布」
2. 复制表单链接
3. 保存备用

## 📊 数据管理

### 查看注册数据

1. 登录金数据后台
2. 点击你的表单
3. 点击「数据」
4. 查看所有注册用户

### 导出数据

1. 点击「导出」
2. 选择格式：Excel 或 CSV
3. 下载到本地
4. 用 Excel 打开分析

| 字段 | 说明 | 用途 |
