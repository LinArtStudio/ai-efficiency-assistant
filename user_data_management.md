## 🔍 如何查看注册用户

### 方法1：Vercel 控制台（推荐）

1. **登录 Vercel**
   - 打开 https://vercel.com
   - 使用 GitHub 账号登录

2. **进入项目**
   - 找到你的后端项目
   - 点击进入

3. **查看数据库**
   - 点击「Storage」或「Database」
   - 查看用户表数据

4. **导出数据**
   - 点击「Export」导出 CSV
   - 用 Excel 打开查看

### 方法2：API 直接查询

```bash
# 获取所有用户（需要管理员权限）
curl -X GET "https://backend-roan-pi-60.vercel.app/api/users" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"

# 获取单个用户
curl -X GET "https://backend-roan-pi-60.vercel.app/api/users/USER_ID" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

## 🔐 数据安全与隐私

### 隐私政策要点

1. **数据收集**
   - 仅收集必要信息
   - 明确告知用途

2. **数据使用**
   - 仅用于产品改进和服务
   - 不会出售给第三方

3. **数据存储**
   - 使用安全的云服务
   - 定期备份

4. **用户权利**
   - 用户可随时删除数据
   - 用户可导出个人数据

### 合规建议

1. **添加隐私政策**
   - 在网站底部添加链接
   - 说明数据收集和使用

2. **用户同意**
   - 注册时勾选同意
   - 保留同意记录

3. **数据删除**
   - 提供删除账号入口
   - 响应删除请求
